# ❄️ Tab Freezer — Менеджер вкладок

**Tab Freezer** — расширение для Google Chrome, которое автоматически выгружает из памяти или закрывает неактивные вкладки, освобождая ресурсы системы. Управляйте вкладками через удобную панель, добавляйте сайты в белый список, временно отключайте заморозку для отдельных доменов и отслеживайте историю событий.

---

## ✨ Возможности

- **Автоматическая заморозка** — вкладки, которые не открывались заданное время, выгружаются из памяти (или закрываются полностью, если включен агрессивный режим).
- **Агрессивный режим** — полностью закрывает вкладки, сохраняя их в отдельный список для последующего восстановления. Освобождает 100 % оперативной памяти.
- **Список замороженных вкладок** — все закрытые вкладки сохраняются с заголовком, URL и иконкой. Открываются одним кликом, удаляются по отдельности или списком целиком.
- **Временные исключения** — добавляйте домены, которые не должны замораживаться на заданный срок (от 15 минут до 1 дня). Исключения действуют до истечения времени.
- **Панель управления** — полноценная страница со вкладками:
  - список замороженных (сохранённых) вкладок;
  - список всех открытых вкладок с возможностью закрыть любую;
  - статистика за всё время (сколько вкладок заморожено, оценка сэкономленной памяти);
  - настройки (время бездействия, защита закреплённых и вкладок со звуком, агрессивный режим, автоматическая очистка старых записей);
  - белый список (постоянные исключения по доменам);
  - управление временными исключениями;
  - журнал событий (все действия расширения).
- **Умная защита** — не трогает активные вкладки, закреплённые (опционально), вкладки со звуком (опционально), системные страницы (chrome://, edge://, about:), а также сайты из белого списка и временных исключений.
- **Экспорт и импорт настроек** — сохраняйте все параметры и статистику в JSON-файл и переносите на другой компьютер.
- **Журнал событий** — подробный лог заморозок, закрытий, восстановлений, изменений настроек и ошибок.
- **Быстрый доступ из попапа** — статистика, кнопка ручной заморозки, добавление временного исключения для текущего сайта, ссылка на панель управления.

## 🚀 Установка

Расширение распространяется в виде исходного кода и устанавливается через режим разработчика Chrome.

### Шаг 1. Скачайте файлы

Скачайте и распакуйте архив с проектом в удобную папку на вашем компьютере. Убедитесь, что в корне папки присутствуют файлы:

- `manifest.json`
- `background.js`
- `popup.html` и `popup.js`
- `dashboard.html` и `dashboard.js`
- папка `icons` с иконками (`snowflake-16.png`, `48.png`, `128.png`).

### Шаг 2. Включите режим разработчика в Chrome

1. Откройте Chrome и перейдите по адресу: `chrome://extensions/`
2. В правом верхнем углу включите **«Режим разработчика»** (Developer mode).

### Шаг 3. Загрузите расширение

1. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked).
2. Выберите папку с распакованными файлами.
3. Расширение появится в списке. Закрепите его на панели инструментов (значок пазла → ❄️ → скрепка).

**Важно:** не удаляйте и не перемещайте папку с расширением — Chrome использует её для работы.

## ⚙️ Настройка

Кликните по иконке ❄️ в браузере, чтобы открыть попап, или нажмите «Панель управления» для доступа ко всем настройкам:

- **Время бездействия** — через сколько минут неактивная вкладка будет заморожена (по умолчанию 15).
- **Защита закреплённых** — не выгружать и не закрывать закреплённые вкладки.
- **Защита со звуком** — не трогать вкладки с аудио.
- **Агрессивный режим** — полностью закрывать вкладки вместо выгрузки из памяти.
- **Автоочистка** — автоматически удалять из списка замороженные записи старше заданного времени.
- **Белый список** — домены, которые никогда не будут заморожены или закрыты.
- **Временные исключения** — домены, которые игнорируются на ограниченное время (добавляются из попапа или панели).

## 🛠 Технологии

- Чистый JavaScript (ES6+)
- HTML5 и CSS3
- Manifest V3
- Хранилище: `chrome.storage.local` и `chrome.storage.session`
- Service Worker как background-скрипт
- API вкладок, алармов, контекстных меню

Расширение не использует сторонних библиотек и работает максимально легко.
