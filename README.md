# ❄️ Tab Freezer — Менеджер вкладок

**Tab Freezer** — мощное расширение для браузеров на базе Chromium, которое автоматически выгружает из памяти или полностью закрывает неактивные вкладки, освобождая ресурсы системы и продлевая время работы от батареи. Управляйте вкладками через удобную панель, добавляйте сайты в белый список, временно отключайте заморозку для отдельных доменов и отслеживайте каждое действие в журнале.

---

## ✨ Возможности

- **Автоматическая заморозка** — вкладки, которые не открывались дольше заданного времени, выгружаются из памяти (или закрываются полностью, если включен режим полной заморозки).  
- **Полная заморозка** — полностью закрывает вкладки, сохраняя их в отдельный список для последующего восстановления. Освобождает 100 % оперативной памяти. Теперь доступна **для системных страниц** (новые вкладки, настройки, `about:blank` и т. д.) с возможностью гибкой настройки.  
- **Защита от ложной потери звука** — вкладка с аудио не будет заморожена в течение 10 секунд после того, как звук прекратился. Идеально для видео, которое иногда прерывается на короткое время.  
- **Список замороженных вкладок** — все закрытые вкладки сохраняются с заголовком, URL и иконкой. Открываются одним кликом, удаляются по отдельности или целиком.  
- **Временные исключения** — добавляйте домены, которые не должны замораживаться на заданный срок (от 15 минут до 1 дня). Исключения действуют до истечения времени.  
- **Панель управления** — полноценная страница с вкладками:
  - список замороженных (сохранённых) вкладок;
  - список всех открытых вкладок с возможностью закрыть любую;
  - статистика за всё время (количество замороженных вкладок, оценка сэкономленной памяти);
  - настройки (время бездействия, защита закреплённых и вкладок со звуком, полная заморозка, автоматическая очистка старых записей);
  - **новый раздел «Настройки полной заморозки»** — разрешает замораживать системные страницы и задавать список URL-префиксов для них;
  - белый список (постоянные исключения по доменам);
  - управление временными исключениями;
  - журнал событий (все действия расширения).
- **Умная защита** — не трогает активные вкладки, закреплённые (опционально), вкладки со звуком (с буфером 10 секунд), критичные системные страницы (`chrome://extensions`, `chrome://settings` и их аналоги), а также сайты из белого списка и временных исключений.  
- **Экспорт и импорт настроек** — сохраняйте все параметры и статистику в JSON-файл и переносите на другой компьютер.  
- **Журнал событий** — подробный лог заморозок, закрытий, восстановлений, изменений настроек и ошибок.  
- **Быстрый доступ из попапа** — статистика, кнопка ручной заморозки, добавление временного исключения для текущего сайта, ссылка на панель управления. Попап теперь открывается **мгновенно** даже после «засыпания» сервис-воркера.

## 🚀 Установка

Расширение распространяется в виде исходного кода и устанавливается через режим разработчика в браузерах на базе Chromium (Chrome, Edge, Opera и др.).

### Шаг 1. Скачайте файлы

Скачайте и распакуйте архив с проектом в удобную папку на вашем компьютере. Убедитесь, что в корне папки присутствуют:

- `manifest.json`
- `background.js` (главный сервис-воркер)
- папки `bg/` и `ui/` с модулями (все файлы необходимы для работы)
- `popup.html` и `popup.js`
- `dashboard.html`, `dashboard.css` и `dashboard.js`
- папка `icons` с иконками (`snowflake-16.png`, `48.png`, `128.png`).

### Шаг 2. Включите режим разработчика

1. Откройте браузер и перейдите на страницу расширений:  
   - Chrome: `chrome://extensions/`  
   - Edge: `edge://extensions/`  
2. В правом верхнем углу включите **«Режим разработчика»** (Developer mode).

### Шаг 3. Загрузите расширение

1. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked).
2. Выберите папку с распакованными файлами.
3. Расширение появится в списке. Закрепите его на панели инструментов (значок пазла → ❄️ → скрепка).

**Важно:** не удаляйте и не перемещайте папку с расширением — браузер использует её для работы.

## ⚙️ Настройка

Кликните по иконке ❄️ в браузере, чтобы открыть попап, или нажмите «Панель управления» для доступа ко всем настройкам:

- **Время бездействия** — через сколько минут неактивная вкладка будет заморожена (по умолчанию 15).
- **Защита закреплённых** — не выгружать и не закрывать закреплённые вкладки.
- **Защита со звуком** — не трогать вкладки с аудио (с умным буфером 10 секунд после потери звука).
- **Полная заморозка** — полностью закрывать вкладки вместо выгрузки из памяти (сохраняются в списке «Замороженные»).  
- **Полная заморозка системных страниц** — разрешает применять полную заморозку к системным страницам (`chrome://newtab`, `about:blank` и др.). Можно указать конкретные URL-префиксы, которые нужно закрывать, или оставить список пустым — тогда будут замораживаться **все** системные страницы (кроме критических, которые защищены всегда).
- **Автоочистка** — автоматически удалять из списка замороженные записи старше заданного времени.
- **Белый список** — домены, которые никогда не будут заморожены или закрыты.
- **Временные исключения** — домены, которые игнорируются на ограниченное время (добавляются из попапа или панели).

## 🛠 Технологии

- Чистый JavaScript (ES-модули)  
- HTML5 + CSS3  
- Manifest V3  
- Хранилище: `chrome.storage.local` с защитой от конкурентных записей (мьютекс) и автоматическим освобождением квоты  
- Service Worker как background-скрипт (с долгоживущим соединением для поддержания активности)  
- API вкладок, алармов, контекстных меню  
- Модульная архитектура: бизнес-логика (`bg/`), пользовательский интерфейс (`ui/`), общие утилиты (`shared.js`)

Расширение не использует сторонних библиотек и работает максимально легко.
