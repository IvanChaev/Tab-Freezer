# ❄️ Tab Freezer — Менеджер вкладок

**Tab Freezer** — расширение для браузеров на базе Chromium, которое автоматически выгружает из памяти или полностью закрывает неактивные вкладки. Это освобождает ресурсы системы, продлевает время работы от батареи и помогает держать браузер в порядке. Управление — через удобную панель, белый список доменов, временные исключения и подробный журнал.

---

## ✨ Возможности

- **Автоматическая заморозка** — вкладки, которые не использовались дольше заданного времени, выгружаются из памяти (или закрываются полностью в режиме полной заморозки).  
- **Полная заморозка** — полностью закрывает вкладки, сохраняя их в отдельный список для восстановления в один клик. Память освобождается на 100 %.  
  Теперь можно выборочно замораживать **системные страницы** (`chrome://newtab`, `about:blank` и другие) — достаточно включить опцию и указать нужные адреса или их начала. Пустой список означает, что системные страницы не будут заморожены (защита от случайного закрытия важных служебных страниц).  
- **Умный учёт активности** — расширение запоминает, когда вкладка последний раз была активна, даже после перезапуска браузера. При старте браузера таймеры неактивности сбрасываются, чтобы старые вкладки не замораживались сразу.  
- **Защита от ложной потери звука** — если вкладка воспроизводила звук, она не будет заморожена в течение 10 секунд после его отключения (удобно для видео с короткими паузами).  
- **Список замороженных вкладок** — все закрытые вкладки сохраняются с названием, адресом и иконкой. Открываются одним кликом, удаляются по одной или все сразу.  
- **Временные исключения** — домены, которые не должны замораживаться в течение заданного времени (от 15 минут до суток).  
- **Панель управления** — полноценная страница с разделами:
  - замороженные вкладки (можно искать, сортировать, открывать);
  - все открытые вкладки (можно закрыть любую, добавить временное исключение);
  - статистика (количество замороженных вкладок и приблизительная экономия памяти);
  - настройки (время бездействия, защита закреплённых и вкладок со звуком, полная заморозка, автоочистка старых записей);
  - **настройки полной заморозки системных страниц** (включение и список разрешённых адресов);
  - белый список (постоянные исключения по доменам);
  - временные исключения (просмотр и удаление);
  - журнал событий.
- **Удобные мелочи** — счётчики в боковом меню всегда показывают общее количество вкладок, а не только найденное при поиске; таймеры обновляют данные, не сбивая прокрутку; при переключении между вкладками браузера панель автоматически обновляет список; есть уведомление о завершении заморозки и кнопка «Повторить» при ошибках загрузки.  
- **Расширенная поддержка браузеров** — корректно распознаёт системные страницы не только в Chrome и Edge, но и в Opera, Brave, Vivaldi, Arc, Whale и других браузерах на Chromium.  
- **Экспорт и импорт настроек** — сохраните все параметры и статистику в JSON-файл, чтобы перенести на другой компьютер.  
- **Журнал событий** — подробная история заморозок, восстановлений, изменений настроек и ошибок.  
- **Быстрый доступ из попапа** — статистика, кнопка «Заморозить неактивные», добавление временного исключения для текущего сайта, ссылка на панель управления. Попап открывается мгновенно.

## 🚀 Установка

Расширение устанавливается через режим разработчика в браузерах на базе Chromium (Chrome, Edge, Opera и др.).

### Шаг 1. Скачайте [новейшую версию](https://github.com/IvanChaev/Tab-Freezer/archive/refs/tags/v3.5.0.zip)

Скачайте и распакуйте архив с проектом в удобную папку.


### Шаг 2. Включите режим разработчика

1. Откройте страницу расширений:  
   - Chrome: `chrome://extensions/`  
   - Edge: `edge://extensions/`  
2. В правом верхнем углу включите **«Режим разработчика»**.

### Шаг 3. Загрузите расширение

1. Нажмите **«Загрузить распакованное расширение»**.
2. Выберите папку с файлами.
3. Расширение появится в списке. Закрепите его на панели инструментов (значок пазла → ❄️ → скрепка).

> **Важно:** не удаляйте и не перемещайте папку с расширением — браузер использует её для работы.

## ⚙️ Настройка

Кликните по иконке ❄️ в браузере, чтобы открыть попап, или выберите «Панель управления» для доступа ко всем параметрам:

- **Время бездействия** — через сколько минут неактивная вкладка будет заморожена (минимум 1 минута, по умолчанию 15).
- **Защита закреплённых** — не трогать закреплённые вкладки.
- **Защита со звуком** — не трогать вкладки с аудио (с буфером 10 секунд после потери звука).
- **Полная заморозка** — полностью закрывать вкладки вместо выгрузки из памяти (сохраняются в списке «Замороженные»).  
- **Полная заморозка системных страниц** — разрешает закрывать системные страницы. Чтобы это работало, **необходимо добавить хотя бы один адрес или префикс** в список. Пустой список = системные страницы не замораживаются. Критически важные страницы (например, `chrome://extensions`) защищены всегда.
- **Автоочистка** — автоматически удалять из списка замороженные записи старше заданного времени.
- **Белый список** — домены, которые никогда не будут заморожены или закрыты.
- **Временные исключения** — домены, которые игнорируются на ограниченное время (добавляются из попапа или панели).

## 🛠 Технологии

- Чистый JavaScript (ES-модули)  
- HTML5 + CSS3  
- Manifest V3  
- Хранилище: `chrome.storage.local` с защитой от конфликтов и автоматическим освобождением квоты  
- Надёжный фоновый процесс, который корректно обрабатывает остановки и перезапуски  
- Модульная архитектура: бизнес-логика (`bg/`), интерфейс (`ui/`), общие утилиты (`shared.js`)

Расширение не использует сторонних библиотек и работает максимально легко.
